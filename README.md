**Bugtopia Plus**

**简介：**

觉得培养虫虫的过程太漫长？想要快速验证变异路线？还是只想轻松享受养成的乐趣？

本 Mod 将多种节省时间的功能整合在一起，极大加快游戏节奏（但也减少了游玩时长），支持部分自定义配置。

**核心功能：**

✅ **光速升级**： 默认提供 **10倍** 的食物经验获取速率，让你的虫虫飞速成长。（可在配置中调整倍率）

✅ **极速繁育**： 默认提供 **10倍** 的交配速度和卵孵化/生长速度，告别漫长的等待。（可在配置中调整倍率）

✅ **掌控变异**： 默认将变异概率调整为 **10%**，让闪光不再是传说。（可在配置中调整概率）

✅ **解除限制**： 解除了蛹阶段不可喂食的限制，最大化利用每一分钟。

**【安装指南】**

1.  **安装 BepInEx**： 如果你是第一次使用 Mod，请先从 [BepInEx的GitHub发布页](https://github.com/BepInEx/BepInEx/releases) 下载最新版本的 BepInEx。
2.  **解压**： 将压缩包内的所有文件，解压到《虫虫生态箱》的游戏根目录 (也就是与 `Bugtopia.exe` 文件放在一起)。
3.  **安装本 Mod**： [下载 BugtopiaPlus.dll](https://github.com/KittyClub-Yuxiu/Bugtopia-BugtopiaPlus/releases)。
4.  **放入插件文件夹**： 将 `BugtopiaPlus.dll` 文件，放入游戏根目录下的 `BepInEx/plugins` 文件夹内。(如果 plugins 文件夹不存在，请先运行一次游戏，它会自动生成)。
5.  **完成！** 启动游戏即可享受！

**【如何修改配置】**

1.  先运行一次游戏，让 Mod 生成配置文件，随后退出游戏。
2.  进入 `BepInEx/config` 文件夹。
3.  找到名为 `BugtopiaPlus.cfg` 的文件，用记事本打开。
4.  修改等号后面的数值（例如将 `FoodExpMultiplier = 10` 改为 `2`），保存并重启游戏即可生效。
